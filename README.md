# Notification Widget(ReactJs)
 
## Install required dependencies

    npm install

## Run the app

    npm start

## URL to access the application 

    http://localhost:3000/

## Configuration 

   User will not be able to see details If an Item has `clickable:false`, this is being served in server response.
